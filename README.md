Requirement

python 3.9.17

pytorch 1.13.1

mmdet3.1.0

mmcv 2.0.1

具体安装教程参见[mmdetection官网教程](https://mmdetection.readthedocs.io/en/v3.1.0/get_started.html)

想要运行AO-DETR只需要运行tools/train_ao-detr.py

注意，你需要自己下载resnet50模型，以及Swin-L模型
